# Generate Marketing Ad Banners with LINE, Gemini, and Nano Banana Pro

Advanced n8n automation for Generate Marketing Ad Banners with LINE, Gemini, and Nano Banana Pro.

## Overview
- Category: Content Creation, Multimodal AI
- Complexity: advanced
- Source: n8n workflow template export

## What This Automation Does
Effortlessly turn LINE messages into high-quality, text-rich advertising banners for the Japanese market using AI and Google Gemini. Learn more now!

## Included Files
- `workflow.json`

## Setup
1. Import `workflow.json` into n8n.
2. Configure required credentials for the services used in the workflow nodes.
3. Update any environment variables or static values inside nodes (API keys, URLs, IDs).
4. Run a test execution and then activate the workflow.
